## Shortcuts

### UI

```
⌘⌥T   Close all tabs except current
⌘⌥→   Go to right (left) tab
⌘⇧E   Open project explorer
⌘R    Toggle right sidebar
⌘B    Toggle left sidebar
```

### Basic editing

```
⌥↑    Move line up (down)
⇧⌥↑   Copy line up (down)
⌘↵    Insert line below
⇧⌘↵   Insert line above
⇧⌘\   Jump to matching bracket
⌘]    Indent line
⌘[    Outdent line
⌘/    Toggle line comment
```

### Multi-cursor and selection

```
⌥•    Insert cursor
⌥⌘↑   Insert cursor above (below)
⌘U    Undo last cursor operation
```

### Search and replace

```
⌘F      Find
⌥⌘F     Replace
⌘D      Add selection to next find match
⌘K ⌘D   Move last selection to next find match

```
